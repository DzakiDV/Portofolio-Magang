Tentang Saya

Saya siswa yang saat ini sedang melakukan kegiatan Magang/PKL di Gamelab Indonesia pada bidang pengembangan web. Selama magang di GameLab Indonesia, saya belajar dan mengembangkan keterampilan dalam frontend web development, termasuk pembuatan antarmuka pengguna yang responsif dan interaktif. Saya percaya bahwa desain web yang baik dapat memberikan pengalaman yang bermakna dan memuaskan bagi pengguna.

Beberapa Poyek yang Sudah Saya Selesaikan

Project 1 : Landing Page UMKM Riffanash Catering

Landing page ini dirancang untuk UMKM penyedia catering Riffanash Catering, yang menonjolkan keahlian mereka dalam menyajikan masakan Nusantara. Halaman ini mencakup Beranda, Tentang Riffanash, Menu Makanan/Catering, FAQ, dan Kontak. Desainnya responsif, memastikan pengalaman pengguna yang optimal di berbagai perangkat, baik desktop, tablet, maupun ponsel.

Project 2 : Landing Page UMKM Arsi Coffee Shop

Landing page ini dibuat untuk UMKM coffee shop Arsi, yang berfokus pada promosi produk kopi unggulan mereka. Halaman ini mencakup Beranda, Tentang Arsi, Daftar Produk, Cara Pesan, FAQ, dan Testimoni. Dengan desain modern yang sepenuhnya responsif, web ini memberikan pengalaman pengguna yang optimal di berbagai perangkat, memastikan kemudahan pelanggan dalam menjelajahi dan memesan produk.

Project 3 : Web Company Profile Pixelique Studio

Web company profile ini dibuat sebagai studi kasus untuk perusahaan fiksi Pixelique Studio yang bergerak di bidang kreatif. Website ini mencakup Home Page, About, Services, Portfolio, Contact, dan Testimonial. Dengan desain profesional dan sepenuhnya responsif, web ini dirancang untuk menampilkan identitas perusahaan serta portofolio layanan kreatif mereka secara menarik di berbagai perangkat.
